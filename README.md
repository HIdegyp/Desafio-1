# Desafio-1 — Results Summary Component

Componente de cartão de resultados desenvolvido com HTML e CSS puro, sem frameworks ou bibliotecas externas.

## Sobre o projeto

Exibe um cartão dividido em dois painéis:

- **Your Result** — mostra a pontuação geral do usuário em um círculo com gradiente e uma classificação textual (ex: "Great").
- **Summary** — lista as categorias avaliadas (Reaction, Memory, Verbal e Visual) com as respectivas pontuações.

## Tecnologias

- HTML5
- CSS3 (Flexbox, variáveis CSS, gradientes)

## Estrutura de arquivos

```
index.html
css/
  reset.css       # reset de estilos padrão do navegador
  variables.css   # variáveis de cor e tipografia
  base.css        # estilos globais e de layout
  card.css        # estilos específicos do componente de cartão
images/
  icon-reaction.svg
  icon-memory.svg
  icon-verbal.svg
  icon-visual.svg
  favicon-32x32.png
```

## Como executar

Basta abrir o arquivo `index.html` diretamente no navegador — não há dependências ou etapa de build necessária.
